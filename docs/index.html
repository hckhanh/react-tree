react-tree-es6

jsTree is written on library, my wrapper just wrap it to make sure it can work well with life cycle of React. All APIs from jsTree are apply to this wrapper. So you can work with this component like jsTree.

For now, it' just has some basic features. If you think there are some better ideas can apply to this wrapper, please make a issue. Any help would be appreciated.
